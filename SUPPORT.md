# Support

## Where To Ask Questions

- Usage questions: open a GitHub Discussion or Issue in this repository.
- Bug reports: use the bug issue template.
- Feature requests: use the feature request template.

## Security Issues

Do not report potential vulnerabilities publicly.

Follow the private reporting guidance in `SECURITY.md`.

## What To Include

When requesting help, include:

- PDV version (`pdv --version`)
- OS and shell
- Reproduction steps
- Relevant logs/error output
